# seer-prediction-markets

OpenClaw skill for interacting with [Seer](https://seer.pm) prediction markets. Create, trade, provide liquidity, and resolve markets across Gnosis, Base, Optimism, and Mainnet.

## Installation

```bash
cd ~/.openclaw/skills
git clone https://github.com/greenlucid/seer-prediction-markets.git
```

That's it. Ask your agent to check out markets on Seer and start creating markets!

## What's included

- **Trading**: Buy/sell outcome tokens, manage positions, redeem winners
- **Market creation**: Categorical, scalar, and multi-scalar markets
- **Liquidity provision**: Concentrated liquidity with probability-based ranges
- **Resolution**: Answer Reality.eth questions, resolve markets
- **Multi-chain**: Gnosis, Base, Optimism, Mainnet

See [SKILL.md](SKILL.md) for full documentation.
